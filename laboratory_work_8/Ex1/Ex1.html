Свойства CSS

  • list-style-type - устанавливает вид маркера
  • transition - Позволяет делать плавные переходы между двумя значениями
  • color - определяет цвет текста элемента
  • z-index - Определяет порядок позиционированных элементов по z-оси
  • box-shadow - Добавляет тень к элементу
